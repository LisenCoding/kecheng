## 功能概览
>课程管理系统包括学生、教师、管理员三个模块
>
>:star:学生功能：主要包括选课、查看已选课程列表、成绩查看、课表查询、个人信息管理等
>
>:star:教师功能：主要包括授课查询、课表查询、成绩录入等
>
>:star:管理员功能：系管理、专业管理、班级管理、教师管理、学生管理、课程管理、选课管理等
